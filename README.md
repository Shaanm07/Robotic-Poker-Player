## Overview
This project drives a VEX robot to deal cards and run game actions, intended for a physical poker-table setup.
It is able to handle between 2 and 7 players. 

## Getting started
1. Open `main.iqcpp` in the appropriate VEX IDE for IQ C++.
2. Alternatively, open 'main.cpp' and paste all code in the IQ C++ IDE.
3. Connect the robot brain.
4. Build + download to hardware.

Note: This project requires exact robot setup to function as intended.
